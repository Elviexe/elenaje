專題作品展示

經紀公司管理系統

課程：網際網路與應用

學習 Web API 建置與響應式網站設計 模擬經紀公司藝人管理網站：以 ASP.NET Core MVC + Bootstrap 製作前後端，並串接 SQL 資料庫（SSMS），實現 登入/登出、註冊、權限、藝人資料管理與查詢功能

技術與工具

C#、ASP.NET Core MVC、Bootstrap、SQL Server Management Studio (SSMS)、Entity Framework Core。

專題成果

建立一個模擬經紀公司藝人管理的網站，實現使用者登入/登出、註冊、權限控管，以及藝人資料的新增、修改、刪除與查詢功能。前端採用 Bootstrap 打造響應式介面，後端使用 C# 搭配 ASP.NET Core MVC，並透過 Entity Framework Core 串接 SQL Server 資料庫，實現完整的資料管理功能。

操作說明

使用者可先註冊帳號並登入系統，進入後可依權限瀏覽或管理藝人資料。管理者可新增、修改或刪除藝人資料；一般使用者則可進行藝人資訊查詢與瀏覽。
